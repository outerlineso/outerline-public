# Outerline

This repository is the public home for issue tracking, feature requests, and discussions for **Outerline**.

Outerline itself is closed-source, so this repo does not contain application code. Instead, it’s the place to report bugs, suggest improvements, ask questions, and follow along with what’s coming next.

## What this repo is for

* Bug reports and reproducible issues
* Feature requests and product feedback
* General questions and discussions
* Roadmap notes and known issues

## What this repo is not

* Source code for the app
* Technical support via pull requests
* A guarantee that every request will be implemented

## How to get help

* **Bugs / feature requests** → Open an Issue
* **Questions or discussion** → Start a Discussion
* **Security concerns** → Please don’t post publicly. See `SECURITY.md` for reporting instructions.

When reporting bugs, including your macOS version, Outerline version, and clear repro steps helps a lot.
