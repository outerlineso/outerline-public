# Outerline Public

This repository is the public home for Outerline bug reports, feature requests, support questions, and roadmap discussion.

The Outerline application source code lives in a separate private repository. This public tracker is intentionally lightweight so people can report issues and follow product-facing work without exposing the private codebase.

## Use this repository for

- Bug reports
- Feature requests
- Support questions that are safe to discuss publicly
- Documentation gaps
- Public roadmap discussion

## Do not post

- Private markdown documents or vault contents
- Credentials, tokens, or personal data
- Full crash logs or file paths that reveal private information
- Implementation details from the private code repository
