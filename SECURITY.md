# Security Policy

Please do not report security issues in public GitHub issues.

If you believe you have found a security or privacy issue in Outerline, open a private report through GitHub's security advisory flow when available, or contact the project maintainers privately before sharing details publicly.

When reporting, include:

- A short description of the issue
- Steps to reproduce it
- Affected platform and Outerline version
- Whether private documents, filesystem permissions, or network access are involved
