# Contributing to Outerline public tracking

Thanks for helping shape Outerline. This repository is for public issue tracking, feature requests, and support conversations. The application source code lives in a separate private repository.

## Good issues include

- What you were trying to do
- What happened instead
- Your platform and Outerline version
- Steps that reproduce the problem, if it is a bug
- Screenshots or short screen recordings when they clarify the behavior

## Please avoid posting

- Private documents or vault contents
- Personal access tokens, API keys, or crash logs containing sensitive paths
- Duplicate reports without new detail

## Scope

Use this repository for public product feedback, bugs, feature requests, documentation gaps, and support triage. Internal implementation work remains in the private code repository.
