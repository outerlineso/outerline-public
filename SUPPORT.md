# Support

Use GitHub Issues in this repository for public bugs, feature requests, and documentation gaps.

Before opening an issue, please check whether a similar report already exists. If your report includes private document content, account details, or other sensitive information, do not post it publicly.

## Useful details

- Outerline version and build number
- macOS or iOS version
- Whether your vault is local, iCloud Drive, or another synced folder
- Steps to reproduce the issue
- What you expected to happen
- What actually happened

## Private information

Do not attach private markdown files, credentials, or logs that include sensitive data. Summarize the behavior instead, or redact carefully before sharing.
